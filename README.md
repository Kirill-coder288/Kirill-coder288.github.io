# NAILÉ

Адаптивный лендинг мастера маникюра с премиальной журнальной стилистикой и мягкими анимациями.

## Что внутри

- адаптивный первый экран с лёгким параллаксом;
- услуги, портфолио с автопрокруткой, отзывы и FAQ;
- плавные появления блоков и деликатные hover-эффекты;
- форма записи с готовым фронтенд-сценарием;
- поддержка `prefers-reduced-motion`;
- Open Graph-карточка и русскоязычные метаданные.

## Запуск

Требуется Node.js 22.13 или новее.

```bash
npm ci
npm run dev
```

Проверка production-сборки:

```bash
npm run build
npm test
```

Перед боевым запуском подключите форму в `app/page.tsx` к Telegram, CRM или выбранному сервису онлайн-записи и замените демонстрационные имя, цены и тексты на реальные данные мастера.

# NAILÉ

Адаптивный лендинг мастера маникюра с премиальной журнальной стилистикой и мягкими анимациями.

## Что внутри

- адаптивный первый экран с лёгким параллаксом;
- актуальный каталог услуг, адаптивное портфолио, стандарты сервиса и FAQ;
- плавные появления блоков и деликатные hover-эффекты;
- глубокие ссылки в Telegram Mini App, включая предвыбор конкретной услуги;
- поддержка `prefers-reduced-motion`;
- Open Graph-карточка, robots.txt, sitemap, manifest и структурированные данные.

## Запуск

Требуется Node.js 22.13 или новее.

```bash
pnpm install --frozen-lockfile
pnpm dev
```

Проверка production-сборки:

```bash
pnpm build
pnpm test
```

Перед коммерческим запуском замените демонстрационные фотографии и сведения о мастере на материалы, права на которые подтверждены. Кнопки записи уже открывают `@nogti000bot`; цены и длительность должны поддерживаться синхронно с каталогом Mini App.
